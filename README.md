## Development Setup

requirement:

- rust >= 1.75.0
- nodejs >= v21.5.0
- [just](https://github.com/casey/just)

### Running Step

1. for first time setup run: `just install`
2. copy the `build.env.sample` as `build.env`
3. build and run the app: `just run`
